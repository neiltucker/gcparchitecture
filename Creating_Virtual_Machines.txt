### Creating Virtual Machines
### These exercises should be done only in QWIK Labs

Create two instances.  One using the Console and the other using the Cloud Shell.  Both machines must meet the following criteria:

1.  Both machines must be in the same Region but different zones

2.  They must both use the machine type: n1-highmem-2

3.  Operating System:  Ubuntu

4.  Persistent Disk Size:  40 GB

5.  Firewall:  Allow both HTTP and HTTPS access

6.  Preemptibility:  ON

7.  Delete boot disk when instance is deleted:  OFF

8.  Verify that instance settings and that the VMs can communicate with each other

9.  What will happen to the boot disks when these instances are deleted?
